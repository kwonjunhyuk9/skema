Math

기본 수학 연산

숫자의 절대값, 반올림, 올림, 내림을 처리하는 함수들
Math.abs(-5);          // 5 (절대값)
Math.round(4.7);       // 5 (반올림)
Math.ceil(4.3);        // 5 (올림)
Math.floor(4.7);       // 4 (내림)

최대/최소값

여러 숫자 중 최대값과 최소값을 찾는 함수들
Math.max(1, 2, 3);     // 3 (최대값)
Math.min(1, 2, 3);     // 1 (최소값)

거듭제곱과 제곱근

숫자의 거듭제곱과 제곱근을 계산하는 함수들

Math.pow(2, 3);        // 8 (2의 3제곱)
Math.sqrt(16);         // 4 (제곱근)

삼각함수

기본적인 삼각함수 계산을 위한 함수들

Math.sin(Math.PI / 2); // 1 (사인)
Math.cos(0);           // 1 (코사인)
Math.tan(Math.PI / 4); // 1 (탄젠트)

랜덤값

0과 1 사이의 난수를 생성하는 함수

Math.random();         // 0 ~ 1 사이의 난수

수학 상수

자주 사용되는 수학적 상수들

Math.PI;               // 3.141592653589793 (원주율)
Math.E;                // 2.718281828459045 (자연로그의 밑)
